# 校园招聘系统项目说明文档

## 项目概述

校园招聘系统是一个基于Spring Boot + Vue.js的前后端分离项目，旨在为高校学生和企业提供一个便捷的招聘平台。系统包含三个主要模块：管理端、企业端和用户端。

## 技术架构

### 后端技术栈
- **框架**: Spring Boot 2.5.3
- **数据库**: MySQL 8.0
- **ORM**: MyBatis Plus 3.5.3.1
- **安全**: Spring Security + JWT
- **缓存**: Redis
- **文档**: Swagger 3.0.0
- **文件处理**: Apache POI 5.2.4
- **邮件**: Spring Boot Mail
- **WebSocket**: Spring WebSocket

### 前端技术栈
- **框架**: Vue.js 2.6.14
- **UI组件**: Element UI 2.15.6
- **路由**: Vue Router 3.5.1
- **状态管理**: Vuex 3.6.2
- **HTTP客户端**: Axios 0.24.0
- **图表**: ECharts 6.0.0

## 核心功能模块

### 1. 用户管理模块
- 用户注册/登录
- 个人信息管理
- 角色权限控制（普通用户、企业用户、管理员）
- 密码修改

### 2. 企业管理模块
- 企业注册审核
- 企业信息管理
- 企业状态管理

### 3. 职位管理模块
- 职位发布
- 职位审核
- 职位搜索筛选
- 职位详情展示

### 4. 求职申请模块
- 简历投递
- 申请状态跟踪
- 企业回复管理

### 5. 简历管理模块
- 简历上传（支持PDF、DOC、DOCX）
- 简历预览
- 简历管理

### 6. 反馈管理模块
- 用户反馈提交
- 反馈处理

### 7. 文章管理模块
- 文章发布
- 文章评论
- 文章收藏

## 数据库设计

### 主要数据表

1. **user** - 用户表
   - 存储用户基本信息、角色、积分等

2. **company** - 公司表
   - 存储企业信息、审核状态等

3. **employment** - 职位表
   - 存储职位信息、薪资、要求等

4. **employment_user** - 求职申请表
   - 存储用户投递记录、回复状态等

5. **user_resume** - 用户简历表
   - 存储简历文件信息

6. **feedback** - 反馈表
   - 存储用户反馈信息

7. **article** - 文章表
   - 存储文章内容、点赞收藏等

## 系统特色

### 1. 多角色权限管理
- 普通用户：浏览职位、投递简历、管理个人信息
- 企业用户：发布职位、查看简历、回复申请
- 管理员：审核企业、管理职位、系统维护

### 2. 智能简历处理
- 支持多种格式简历上传
- 自动转换DOC/DOCX为HTML预览
- 简历文件管理

### 3. 实时通信
- WebSocket实现实时消息推送
- 申请状态实时更新

### 4. 安全机制
- JWT Token认证
- Spring Security权限控制
- 密码加密存储
- CORS跨域处理

## 部署说明

### 环境要求
- JDK 8+
- MySQL 8.0+
- Redis
- Node.js 14+

### 后端部署
1. 配置数据库连接（application.yml）
2. 导入数据库脚本
3. 启动Redis服务
4. 运行Spring Boot应用

### 前端部署
1. 安装依赖：`npm install`
2. 开发环境：`npm run serve`
3. 生产构建：`npm run build`

## 接口说明

系统提供RESTful API接口，支持：
- 用户认证与授权
- 数据CRUD操作
- 文件上传下载
- 分页查询
- 条件筛选

详细API文档请参考Swagger文档：`http://localhost:3030/swagger-ui/`

## 项目亮点

1. **前后端分离架构**：提高开发效率和系统可维护性
2. **多端适配**：支持管理端、企业端、用户端
3. **权限精细化控制**：基于角色的访问控制
4. **文件处理能力**：支持多种格式简历处理
5. **实时通信**：WebSocket实现消息推送
